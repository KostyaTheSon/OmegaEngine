С помощью этой утилиты создаются файлы которые служат основой для шрифта которые понимает движок. Файл сохраненный вами, в формате bmp, представляет собой текстуру будущего шрифта, а созданный рядом файл с расширением ofi - это двоичный файл несущий информацию о самом шрифте(ширина символов и т.д.).

Итак, приступим к рассмотрению возможностей:

Кнопка Save to file сохраняет в файлы нужные данные.
Кнопка Width+ открывает выбранный вами ofi-файл(!) и изменяет ширину всех символов на указанное значение. Это потребуется если вы поверх созданного шрифта наложили эффект и хотите чтоб текст не смотрелся обрезанным по бокам. Все остальное должно быть интуитивно понятно.

О, и маленький совет - стандартные шрифты очень удобно хранить в gif-формате=)
